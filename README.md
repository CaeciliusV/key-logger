# Python Keyloggers

## Status: In Progress

## Reason for series
I created this series so that i can learn about Key logging, how it works, the dangers of it and how to counter it. This is to give me experience of the attacks that happen in the real world alongside my T Level.

## Versions
V1 - Simple keylogger that can record the keystrokes of the device it is running on and write them instantly onto a txt file.
V2 - Networked keylogger that can record the keystrokes of a client device and every 30 seconds those keystrokes are sent to the server device that is on the same network. Once the server device receives it, the keystrokes are then written onto a txt file on the server device.
